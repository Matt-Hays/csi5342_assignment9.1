## Quick Run

```bash
# 1 · install Ceedling (once per machine)
gem install ceedling           # needs Ruby 2.7+

# 2 · clone & enter the project
git clone https://github.com/your-org/microinfusion_pump_controller.git
cd microinfusion_pump_controller

# 3 · build and run all unit tests
ceedling test:all
```
